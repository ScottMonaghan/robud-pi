# robud-pi
Ro-Bud - Lovable, Accessible, Autonomous Companion (on the Raspberry Pi!)

# Set-up Steps (draft outline)
- Install Mosquitto
- Install paho.mqtt
- Install OpenCV
- Install TendorFlow 2
- Install eSpeak-ng
- Install Librosa
- Install MyCroft Precise
- Install Coqui STT
- Install Blinka
- Install adafruit_motorkit
- Install adafruit_servokit
- Install pyaudio
- Install pytweening
